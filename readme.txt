Simple plugin that enables simple white label customization, primarily to the login page.

== Description ==

Using [Advanced Custom Fields Pro](https://www.advancedcustomfields.com/pro/) and [Advanced Custom Fields Extended](https://www.acf-extended.com/), this plugin will add an option page under the Appearance admin menu. From there, the image and image height can be customized. Image width is set to `100%`.

== Installation ==

Please ensure [Advanced Custom Fields Pro](https://www.advancedcustomfields.com/pro/) and [Advanced Custom Fields Extended](https://www.acf-extended.com/) are installed and activated before using this plugin.
